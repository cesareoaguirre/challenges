Esta es una solución a un reto de Multiplica Talent.
El requisito es crear una app front-end que pueda facilitar a diseñadores copiar códigos de color de una lista.

Se ocuparon diversos objetos y códigos disponibles en DOM y JavaScript. Para las animaciones no se ocupó ningún framework de terceros.

Es una app contenida en sí misma, funciona abriendo el index.html y no requiere ninguna dependencia, salvo por conectividad con el servicio de JSON provisto por el cliente.
