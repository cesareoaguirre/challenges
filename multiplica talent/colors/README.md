Esta es una solución a un reto de Multiplica Talent.
El requisito es crear una app front-end que pueda facilitar a diseñadores copiar códigos de color de una lista.

Se ocuparon diversos objetos y códigos disponibles en DOM y JavaScript. 


En cuanto al uso de frameworks, en realidad ¡ocupé JavaScript puro!

Se ocupó flex-box.

Dependencias: 
app-colors.html
app-colors.js
app-colors.css


index-self-contained.html
Es una app contenida en sí misma, no requiere ninguna dependencia, salvo por conectividad con el servicio de JSON provisto por el cliente.
